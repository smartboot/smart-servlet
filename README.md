# smart-servlet

![](./pages/public/smart-servlet.svg)

轻量、高性能、面向未来的 Servlet 容器，基于 Jakarta Servlet 6.0，适用于 Java 17+ 环境。

## 为什么选择 smart-servlet？

在国产化替代背景下，企业不仅关注性能，更关注 **自主可控、长期维护以及技术服务能力**。

相比传统 Servlet 容器，smart-servlet 具备以下特点：

* **自主可控**：核心技术完全自主研发，便于企业开展代码审计、安全评估及长期维护；
* **性能优越**：基于最新版 smart-socket 通信微内核构建，具备优秀的并发处理能力；
* **规范兼容**：遵循 Jakarta Servlet 6.0 规范，兼容主流 Servlet 应用开发模式；
* **运行灵活**：支持 War 包、Spring Boot、Maven Plugin 等多种运行方式；
* **服务保障**：提供商业授权、技术支持与长期维护服务。
